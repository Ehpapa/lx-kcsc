# 한국국토정보공사노동조합 홈페이지

POSCO 메인 화면의 풀스크린 영상과 전체 메뉴 전환 방식을 참고해 제작한
한국국토정보공사노동조합 홈페이지 시안입니다.

## 실행

별도 설치 없이 `index.html`을 열거나 로컬 웹 서버로 실행할 수 있습니다.

```powershell
python -m http.server 4173
```

브라우저에서 `http://localhost:4173`으로 접속합니다.

## 배경 영상 교체

현재는 제공된 첫 번째 캡처를 임시 포스터로 사용합니다. 추후 전달받은 MP4
영상을 `assets/hero.mp4`로 저장한 뒤 `index.html`의 `hero-video` 안에 있는
`source` 주석을 해제하면 영상이 자동 재생됩니다.
